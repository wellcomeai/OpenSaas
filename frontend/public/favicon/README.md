# Favicon

Замените файлы в этой папке на свои:

- `favicon.ico` — классическая иконка для старых браузеров
- `favicon-16x16.png`, `favicon-32x32.png`
- `apple-touch-icon.png` (180×180)
- `android-chrome-192x192.png`, `android-chrome-512x512.png`
- `site.webmanifest` — манифест PWA (уже создан, замените `name` если нужно)

Удобный генератор: <https://realfavicongenerator.net>
